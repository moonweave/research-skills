# contradiction-finder

> Status: planned — coming soon.

This skill will ingest a set of papers (by DOI or search query), extract sentence-level claims, compare them pairwise using LLM classification, and output a structured conflict map with verbatim evidence from each source.

See the [monorepo README](../README.md) for the full roadmap.
