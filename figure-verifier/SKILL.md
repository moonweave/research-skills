# figure-verifier

> Status: planned — coming soon.

This skill will extract numerical values from scientific figures via VLM (vision model), identify quantitative claims in the surrounding text, and flag mismatches. The only tool that verifies whether a paper's figures actually support its stated results.

See the [monorepo README](../README.md) for the full roadmap.
